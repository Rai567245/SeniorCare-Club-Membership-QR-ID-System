# 📱 SeniorCare – Club Membership QR ID System

SeniorCare is a web-based membership identification and verification system that utilizes QR code technology to streamline member registration, identification, and validation processes. Built using HTML, CSS, and JavaScript, the application allows organizations, clubs, and community groups to generate unique QR-based membership IDs that can be scanned and verified instantly.

## 🚀 Features

* 📝 Member Registration Form
* 🔐 Unique QR Code Generation
* 📷 Built-in QR Code Scanner
* ✅ Instant Membership Verification
* 📱 Responsive Design for Mobile and Desktop
* ⚡ Browser-Based Operation (No Backend Required)
* 🎫 Digital Membership Identification System

---

## 📋 Member Information Included

Each generated QR code contains the following details:

* Full Name
* Membership Number
* Club Affiliation
* Membership Date
* Expiration Date

## ⚙️ How It Works

1. Enter the member's information.
2. Generate a unique QR code.
3. Save or distribute the generated QR code.
4. Scan the QR code using:

   * The built-in scanner
   * Mobile devices
   * Third-party QR scanner applications
5. Instantly verify the member's information.

---

## 🎯 Purpose

The project demonstrates the practical implementation of QR code generation and scanning technologies for digital membership management. It eliminates the need for physical membership cards while providing a fast, secure, and efficient verification process.
